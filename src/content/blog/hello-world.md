This site used to be a React portfolio. It's now a blog built with [Astro](https://astro.build/).

Posts live as Markdown (or MDX) files in `src/content/blog/`. Frontmatter drives titles, dates and tags.

## Example code

```ts
function greet(name: string) {
  return `Hello, ${name}`;
}

console.log(greet("world"));
```
